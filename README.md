# ClickMyFlector
Winflector AutoStart app
<p>
  This app always starts winflector even when when winflector doesn't always starts the server.
  Just put a shortcut to ClickMyFlector.exe in startup along with winflector server.
</p>
<p>
  All this app does is hunt for Winflector window and send the button click message to the start button
</p>
<p>
  Also if you have Winflector installed in the default folder on the drive where windows is installed use
  command-line parameter /S to have the tool launch winflector with it.
</p>
NOTE IF IT NEVER STARTS WINFLECTOR TRY STARTING WINFLECTOR OUTSIDE THE TOOL AND HAVE THE TOOL FIND
IT THAT WAY
